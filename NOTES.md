# normaliz.wasm — Build Notes

WebAssembly build of [Normaliz](https://github.com/Normaliz/Normaliz) via Emscripten.

## What we have

- Normaliz compiled to wasm (5.2 MB), usable from Node.js and browser
- Emscripten module with virtual filesystem: write `.in` files, call `callMain()`, read `.out` files
- Modularized JS (`createNormaliz()` returns a promise)

## Dependencies cross-compiled to wasm

| Library | Version | Purpose | Notes |
|---------|---------|---------|-------|
| GMP | 6.3.0 | Arbitrary precision (required) | `--host=none --disable-assembly CC_FOR_BUILD=cc` |
| MPFR | 4.2.2 | Multi-precision floats (FLINT dep) | Straightforward cross-compile |
| FLINT | 3.3.1 | Polynomial arithmetic | Had to patch configure to skip assembler label suffix check |
| nauty | 2.9.1 | Automorphism/symmetry | wasm is 32-bit, so we use `nautyW.a` |
| hash-library | 8 | SHA-256 hashing | Trivial single-file build |

Normaliz reports: `with package(s) Flint nauty hash-library`

## Not included

- **CoCoALib** — would enable Ehrhart series integration and Hilbert series simplification. Large C++ library, complex build. Moderate payoff.
- **e-antic** — would enable algebraic number field computations. Niche use case.

## Build quirks

- GMP: emscripten's `configure` couldn't find `HOST_CC`, fixed with `CC_FOR_BUILD=cc`
- FLINT: `configure` fails on "Cannot determine label suffix" even with `--disable-assembly`. Fixed by patching configure with sed to set `gmp_cv_asm_lsym_prefix="L"`.
- Normaliz: uses `NMZ_DEVELOP` flag to skip `version.h` (which is generated by autotools, not present in git checkout). Also uses `NMZ_MAKEFILE_CLASSIC` to skip `nmz_config.h`.
- CMake with emscripten: `find_library`/`find_path` needs `CMAKE_FIND_ROOT_PATH` set to the deps prefix, not just `CMAKE_PREFIX_PATH`.
- nauty: wasm32 means WORDSIZE=32, so the right archive is `nautyW.a` not `nauty.a`.

## Usage (Node.js)

### High-level wrapper (recommended)
```js
const { Normaliz } = require('./src/normaliz.js');
const nmz = await Normaliz.create();
const result = nmz.run('amb_space 2\ncone 2\n1 3\n2 1\n');
console.log(result.output);           // main .out contents
console.log(result.files);            // { out, gen, ext, ... }
console.log(result.console);          // verbose log (if --verbose)

// With flags
const r2 = nmz.run(input, { flags: ['-a', '--verbose'] });

// With real-time console streaming
const nmz2 = await Normaliz.create({
  onStdout: (line) => process.stdout.write(line + '\n')
});
```

### Low-level (direct emscripten module)
```js
const createNormaliz = require('./dist/normaliz.js');
const m = await createNormaliz();
m.FS.writeFile('test.in', 'amb_space 2\ncone 2\n1 3\n2 1\n');
m.callMain(['test']);
const output = m.FS.readFile('test.out', { encoding: 'utf8' });
```
